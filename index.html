HTML

  • The <head> element contains information about the webpage.
  • The <body> element represents the visible content shown to the user.
  • The <header> element contains introductory content and/or a site navigation menu.
  • The <main> element contains the most important content of the page.
  • The <footer> element appears at the bottom of the page and usually contains author, copyright, contact, sitemap, and navigation.
  • When using headings, it is important that we use heading tags in order from <h1> to <h6> to make the page accessible to users who access the page using screen readers or other devices.
  • The <h1> should only appear once on each page.
  • The <alt> attribute for a <img> element contains a text string that describes the appearance and functionality of the image. The alt text is important for screen readers and other assistive technologies. Alt text is known to also improve search engine rankings for your website.

Git

  • git status checks what branch we are currently on.
  • git checkout -b branch-name creates a new branch and switches to it.
  • git add -A add all files to the staging area.
  • git commit -m "commit-message" creates a new version of the of the project that contains all of the files in the staged area.
