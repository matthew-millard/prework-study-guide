Prework Study Guide

HTML

  • The <head> element contains information about the webpage.
  • The <body> element represents the visible content shown to the user.
  • The <header> element contains introductory content and/or a site navigation menu.
  • The <main> element contains the most important content of the page.
  • The <footer> element appears at the bottom of the page and usually contains author, copyright, contact, sitemap, and navigation.
  • When using headings, it is important that we use heading tags in order from <h1> to <h6> to make the page accessible to users who access the page using screen readers or other devices.
  • The <h1> should only appear once on each page.
  • The <alt> attribute for a <img> element contains a text string that describes the appearance and functionality of the image. The alt text is important for screen readers and other assistive technologies. Alt text is known to also improve search engine rankings for your website.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Separation of concerns refers to separating code into different files based on use.
  • External CSS style sheet is the preferred way to style a webpage. Inline CSS and internal style sheets are totally acceptable methods under certain conditions.
  • (DRY) Do Not Repeat Yourself is a coding principle that enourages developers to reuse or share pieces of code in order to reduce the number of lines of code.

Git

  • git status checks what branch we are currently on.
  • git checkout -b branch-name creates a new branch and switches to it.
  • git add -A add all files to the staging area.
  • git commit -m "commit-message" creates a new version of the of the project that contains all of the files in the staged area.

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites.
  • A JavaScript file ends with .js
  • A variable in JavaScript is simply a named container that allows us to store data in our code.
  • To declare a variable in JavaScript, we use the var, const or let keyword. After the keyword, we give the variable a name. This name must be unique because it is how we will reference the information stored in the variable in our code.
  • We use an assignment operator = to assign the specific data that we want stored in our variable. const shape = 'triangle'
  • We can use the console.log() method to output a message to the web console by adding an argument. Example. console.log(shape). This will print the value of the shape variable to the console. // triangle.
  • A string value is contained within quotation marks, ''. It can either be single or double quotations. Just be consistant.
